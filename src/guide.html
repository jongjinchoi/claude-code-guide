터미널 사용 가이드

터미널은 명령어를 입력하여 컴퓨터를 제어하는 프로그램입니다.

터미널 열기

  • Command (Cmd) ⌘ + Space를 누르고 "terminal" 입력
  • Applications → Utilities → Terminal
  • Launchpad에서 "terminal" 검색
터미널을 열면 이렇게 보입니다
Last login: Mon Jan 8 10:32:15 on ttys001 사용자명@MacBook-Pro ~ %

터미널 종료하기

  • Command (Cmd) ⌘ + Q로 완전 종료
  • Command (Cmd) ⌘ + W로 창만 닫기
  • 또는 exit 입력 후 Enter

내용을 확인했다면 완료 버튼을 클릭하세요:

Windows 준비사항

Windows에서 Claude Code를 사용하려면 먼저 시스템 요구사항을 확인해야 합니다.

Windows 버전 확인하기

Win + R을 누르고 winver 입력 후 Enter

버전이 2004 이상인지 확인하세요. (예: 버전 22H2)

💡 Windows에서도 이제 Git for Windows만 설치하면 Claude Code를 사용할 수 있습니다!

Homebrew 설치

5-10분

Homebrew 설치

HomebrewMac용 패키지 관리자. 앱스토어처럼 프로그램을 쉽게 설치할 수 있게 해줍니다.는 Mac에서 프로그램을 설치하는 가장 쉬운 방법입니다.

이 명령어를 터미널에 붙여넣고 Enter를 누르세요:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

💡 비밀번호를 물으면 Mac 로그인 비밀번호를 입력하세요. 화면에 표시되지 않아도 정상입니다.

설치 완료 확인

설치가 완료되면 터미널에서 이 명령어를 실행해보세요:

brew --version

성공하면 이런 메시지가 나타납니다:

터미널 - 성공적인 경우
Last login: Sun Jul 13 10:24:12 on ttys001 사용자명@MacBook-Pro ~ % brew --version Homebrew 4.5.10 Homebrew/homebrew-core (git revision 3d4f5; last commit 2024-01-20) 사용자명@MacBook-Pro ~ %

결과를 선택하세요:

문제 해결

Mac 종류를 선택하세요:

Apple Silicon Mac 해결 방법:

echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"

Intel Mac 해결 방법:

echo 'eval "$(/usr/local/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/usr/local/bin/brew shellenv)"

문제가 해결되었다면:

문제 해결 완료!
Homebrew가 정상적으로 설치되었습니다

Node.js 설치

3분

Node.js 설치

Node.jsJavaScript를 브라우저 밖에서도 실행할 수 있게 해주는 프로그램는 Claude Code가 작동하는 데 필요한 프로그램입니다.

터미널에서 이 명령어를 실행하세요:

brew install node

💡 설치 중에 "Downloading..." 메시지가 나타날 수 있습니다. 잠시 기다려주세요.

설치 완료 확인

설치가 완료되면 이 명령어로 확인해보세요:

node --version

성공하면 이런 메시지가 나타납니다:

터미널 - 성공적인 경우
사용자명@MacBook-Pro ~ % node --version v22.11.0 사용자명@MacBook-Pro ~ %

결과를 선택하세요:

문제 해결

Node.js가 인식되지 않는 경우:

1. 먼저 Node.js 설치 확인:

brew list | grep node

2. 설치되지 않았다면:

brew install node

3. 이미 설치되어 있다면 재설치:

brew reinstall node

4. PATH 설정 문제일 수 있습니다:

💡 Apple Silicon Mac (M1/M2/M3/M4):

echo 'export PATH="/opt/homebrew/bin:$PATH"' >> ~/.zshrc

💡 Intel Mac:

echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

5. 그래도 안 되면 터미널을 완전히 종료하고 다시 열어보세요.

문제가 해결되었다면:

문제 해결 완료!
Node.js가 정상적으로 설치되었습니다

Claude Code 설치

2분

Claude Code 설치

이제 Claude Code를 설치합니다:

npm install -g @anthropic-ai/claude-code

💡 설치 과정에서 여러 패키지를 다운로드합니다. 인터넷 속도에 따라 1-3분이 걸릴 수 있습니다.

설치 완료 확인

설치가 완료되면 이 명령어로 확인해보세요:

claude --version

성공하면 이런 메시지가 나타납니다:

터미널 - 성공적인 경우
사용자명@MacBook-Pro ~ % claude --version @anthropic-ai/claude-code/1.0.56 사용자명@MacBook-Pro ~ %

결과를 선택하세요:

문제 해결

Claude Code가 인식되지 않는 경우:

1. 먼저 설치 확인:

npm list -g @anthropic-ai/claude-code

2. npm 글로벌 경로 확인:

npm config get prefix

3. PATH에 npm 글로벌 경로 추가:

echo 'export PATH="$(npm config get prefix)/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

4. 그래도 안 되면 재설치:

npm uninstall -g @anthropic-ai/claude-code && npm install -g @anthropic-ai/claude-code

5. 터미널을 완전히 종료하고 다시 열어보세요.

문제가 해결되었다면:

문제 해결 완료!
Claude Code가 정상적으로 설치되었습니다

계정 연결

3분

Claude 계정 연결

Claude Code에 Claude 계정을 연결해야 합니다:

claude auth login

💡 브라우저가 열리면서 로그인 페이지가 나타납니다.

브라우저에서 로그인

브라우저가 열리면 Claude 계정으로 로그인하세요.

로그인이 완료되면 터미널로 돌아와서 이 명령어로 확인해보세요:

claude auth status

성공하면 이런 메시지가 나타납니다:

터미널 - 성공적인 경우
사용자명@MacBook-Pro ~ % claude auth Authenticated as: your-email@example.com 사용자명@MacBook-Pro ~ %

결과를 선택하세요:

문제 해결

브라우저가 열리지 않는 경우:

터미널에 표시된 URL을 직접 복사해서 브라우저에 붙여넣으세요.

💡 URL은 보통 https://claude.ai/auth/... 형태입니다.

로그인이 완료되지 않는 경우:

1. 로그아웃 후 다시 시도:

claude-code auth logout
claude auth login

2. 그래도 안 되면 브라우저에서 쿠키를 삭제하고 다시 시도하세요.

계정 관련 문제:

Claude 웹사이트(claude.ai)에서 계정이 정상인지 확인하세요.

문제가 해결되었다면:

문제 해결 완료!
계정이 정상적으로 연결되었습니다

문제 해결

cd ~
mkdir my-project && cd my-project
claude

💡 홈 폴더는 항상 권한이 있으니 거기서 시작하세요!

  • Wi-Fi 확인 - 인터넷 연결되었나요?
  • 브라우저 테스트 - 구글 접속되나요?

  • 터미널 완전히 종료
  • 컴퓨터 재시작
  • 터미널 다시 열고 아래 명령어 실행
cd ~ && claude-code

Microsoft Windows [Version 10.0.19045.3693] (c) Microsoft Corporation. All rights reserved. C:\Users\사용자명>

명령 프롬프트 종료하기

  • Alt + F4로 창 닫기
  • 창의 X 버튼 클릭
  • 또는 exit 입력 후 Enter

Git for Windows 설치

5분

Git for Windows 확인

먼저 Git이 이미 설치되어 있는지 확인합니다:

git --version

버전이 표시되면 다음 단계로 가세요. 오류가 나면 아래 설치 과정을 진행하세요.

Git for Windows 다운로드 및 설치

아래 링크를 클릭하세요:

👉 공식 Git 다운로드 페이지

"Click here to download" 버튼을 클릭하면 자동으로 다운로드가 시작됩니다.

💡 다운로드 완료 후 설치 파일을 실행하고, 모든 화면에서 "Next" 버튼만 클릭하세요. 아무것도 변경하지 마세요!

설치 확인

설치가 완료되면 명령 프롬프트를 다시 열고 확인합니다:

git --version
명령 프롬프트 - 성공적인 경우
C:\Users\사용자명> git --version git version 2.50.1.windows.1

결과를 선택하세요:

문제 해결

명령 프롬프트를 다시 시작했는데도 git을 찾을 수 없는 경우:

  • 1단계: 모든 명령 프롬프트 창을 닫으세요
  • 2단계: 컴퓨터를 재시작하세요
  • 3단계: 명령 프롬프트를 다시 열고 git --version 입력

💡 대부분의 경우 컴퓨터 재시작으로 해결됩니다!

재시작해도 해결되지 않는 경우:

Git을 제거하고 다시 설치하세요:

  • 시작 메뉴 → 설정 → 앱 → Git 찾아서 제거
  • 2단계로 돌아가서 다시 설치
  • 설치 시 모든 기본 옵션 유지 (변경하지 마세요!)

설치 중 오류가 발생하는 경우:

관리자 권한으로 설치 파일을 실행하세요 (마우스 오른쪽 클릭 → "관리자 권한으로 실행")

문제가 해결되었다면:

문제 해결 완료!
Git이 정상적으로 설치되었습니다
